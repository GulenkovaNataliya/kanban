# НАСТРОЙКА SUPABASE ДЛЯ КАНБАН-ДОСКИ

## Шаг 1: Создание проекта

1. Перейдите на https://supabase.com
2. Нажмите "Start your project"
3. Войдите или зарегистрируйтесь
4. Создайте новый проект (Organization → New project)
5. Заполните:
   - Name: kanban-board (или любое имя)
   - Database Password: придумайте надежный пароль
   - Region: выберите ближайший к вам
   - Pricing Plan: Free (достаточно для начала)

## Шаг 2: Создание таблицы

1. В боковом меню выберите "SQL Editor"
2. Нажмите "New query"
3. Вставьте SQL-код, который вам предоставил Supabase (с user_id и авторизацией)
4. Нажмите "Run" для выполнения

✅ Ваш SQL-код уже включает:
- Таблицу tasks с полем user_id
- Row Level Security (RLS)
- Политику безопасности для авторизованных пользователей
- Индексы для оптимизации

## Шаг 3: Получение ключей API

1. В боковом меню выберите "Project Settings" (шестеренка внизу)
2. Перейдите в раздел "API"
3. Скопируйте:
   - **Project URL** (например: https://xxxxx.supabase.co)
   - **anon public** ключ (длинная строка, начинается с eyJ...)

## Шаг 4: Вставка ключей в код

Откройте файл `kanban.html` и найдите в начале JavaScript-секции:

```javascript
const SUPABASE_URL = 'ВАШ_PROJECT_URL_СЮДА';
const SUPABASE_KEY = 'ВАШ_ANON_KEY_СЮДА';
```

Замените на ваши реальные значения из шага 3.

## Шаг 5: Настройка Email Authentication

1. В Supabase перейдите в **Authentication** → **Providers**
2. Убедитесь, что **Email** провайдер включен (он включен по умолчанию)
3. Можете настроить шаблоны писем в разделе **Email Templates**

## Готово!

Теперь канбан-доска работает с авторизацией:
- При первом открытии появится форма входа по email
- Вам придет письмо с magic link для входа
- Каждый пользователь видит только свои задачи
- Изменения синхронизируются в реальном времени!

**Как пользоваться:**
1. Введите свой email и нажмите LOGIN
2. Проверьте почту и кликните на ссылку
3. Вернитесь на канбан-доску - вы автоматически авторизованы
4. Для выхода нажмите кнопку EXIT
